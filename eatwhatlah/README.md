## Business Problem:
Diners often face decision fatigue due to excessive choices, scattered discounts, and unreliable reviews. Current restaurant locator websites lack personalization and real-time feedback. 

Our solution offers personalized recommendations, aggregated deals, and live crowd sentiments to simplify decisions.


Web solution Overview:
## Intended users
Singaporeans who have difficulty chosing a place to eat

## About: 
With an interactive map for nearby dining spots, user tailored food recommendations that aligns with each individual’s tastes, and exclusive discounts compiled from a multitude of sources. The app not only simplifies decision-making but also helps users discover their new favourite food spots , stay ahead of food trends, and enjoy greater value from each meal. In addition, close to real-time emoji updates show how other users feel about each food spot offering an instant, visual snapshot of the crowd’s mood and helping diners make more informed choices.

## Main Features: (do yall wanna add user benefit)
1. Trend-based search algorithm
Search & filter places according to prompts, trends of user preferences and trends in Singapore

2. Real-time reviews
View real-time reviews of nearby food places based on current location and filter (by price, rated delicious by other users,etc.)

3. Real-time map view with user sentiments
View reviews left by other users and their sentiments on the map

4. Discount finder
Displays all available discounts that are filterable based on cuisine, discount type, etc.
